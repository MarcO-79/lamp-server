# lamp-server
This is LAMP Server (PHP 8.4.15 + Apache 2.4 + MySQL 8.0.44 + phpMyAdmin 5.2.3)


# Install
git clone https://github.com/MarcO-79/lamp-server.git

cd lamp-server

# Start
docker-compose up -d

# Stop
docker-compose down

# Runing Server
ip_address:8080 -> Web pages

ip_address:8090 -> phpMyAdmin pages
